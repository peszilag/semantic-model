# Semantic Software and Service Model

This repository contains data models usable by Network Digital Twins
(NDTs) modeling the capabilities of software defined network and
management functions. The data model relies on taxonomies for defining
concepts, which are instantiated to create specific instances of
concepts. The files use YAML for serialization.

**3gpp.yaml**: The taxonomy of 3GPP network functions.

**infra.yaml**: The taxonomy of infrastructure and resources.

**semantic-model.yaml**: The taxonomy (and corresponding JSON schema) of the semantic software and service model.

**deployment.yaml**: The instantiated concepts in a real 5G test network.
